About The App.


This application has been generated using Angular 9.0.2.

The application makes use of Github API in order to display specific information on a username that has been inputted.

The application is able to display a user's avatar picture, username, and a list of repositories that a user currently has has.

The application has been developed as part of a practice on programming concepts.
